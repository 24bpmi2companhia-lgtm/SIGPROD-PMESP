Backup e Segurança

Sistema local. O banco fica salvo no navegador do PC. Faça backup regularmente antes de atualizar arquivos.

Como não perder dados

Os lançamentos ficam salvos no navegador deste computador. Antes de trocar arquivos no GitHub ou testar nova versão, clique em Exportar backup JSON. Se precisar restaurar, use Importar backup JSON.

Como usar no GitHub

Baixe o ZIP, extraia, depois no GitHub clique em Add file → Upload files e envie todos os arquivos desta pasta.
